# Crisis hipertensiva y criterios de derivación segura

## Idea central

Una cifra severamente elevada no se maneja igual si el paciente está asintomático que si presenta síntomas o daño de órgano blanco. El RAG no puede distinguir daño de órgano blanco con certeza, por lo que debe escalar cuando haya síntomas.

## Umbral de preocupación

La AHA marca presión severa cuando sistólica es mayor de 180 mmHg y/o diastólica mayor de 120 mmHg.

## Síntomas de alarma asociados

Dolor torácico, dificultad para respirar, confusión, debilidad/adormecimiento focal, dificultad para hablar, alteración visual, cefalea intensa inusual, síncope, dolor de espalda súbito/desgarrante, disnea intensa, ortopnea o tos con espuma.

## Qué no debe hacer el agente

No recomendar duplicar dosis, no recomendar fármacos específicos, no decir "espere a que baje" si hay síntomas, no tratar ansiedad como explicación suficiente y no sugerir ejercicio para bajar presión en fase aguda.

## Qué sí puede hacer

Pedir repetir medición tras reposo si no hay síntomas graves, verificar técnica de medición, recomendar contacto con profesional y escalar a urgencias si hay síntomas.
